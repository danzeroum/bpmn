Definições de Evento — message/signal/error como cidadãs de primeira classe

Hoje o event definition é inline e anônimo; um export Camunda com messageRef importa como warning/passthrough, não como modelo. Este handoff traz as definições nomeadas (padrão OMG root elements), refs governadas nome@semver via Biblioteca, I/O de eventos na aba Execução, regras de lint EVT_*/TIMER_* e matching honesto no simulador. Pré-requisito: a PR do passthrough zeebe:*/camunda:* mergeada.

⚠ PROBLEMAS — política: lint-etiquette@1.1.0 ● VIGENTE
